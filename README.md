# Two Girls One Mix Website

A complete static website for **Two Girls One Mix**.

## Files

- `index.html` — main website page
- `styles.css` — full responsive styling
- `script.js` — mobile menu behavior
- `assets/two-girls-one-mix-art.png` — cropped brand artwork from the screenshot

## How to run locally

Open `index.html` in your browser.

## How to deploy on Vercel

1. Go to https://vercel.com
2. Create a new project
3. Upload this folder or push it to GitHub
4. Framework preset: **Other**
5. Build command: leave blank
6. Output directory: leave blank
7. Deploy

## What to update before launch

Search inside `index.html` and replace:

- `hello@twogirlsonemix.com`
- `(555) 555-5555`
- `https://formspree.io/f/your-form-id`
- Package pricing if needed
- Service area/location
